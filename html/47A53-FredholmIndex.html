Fredholm index


Let P be a Fredholm operator. The index of P is defined as

index(P) = dim⁡ker⁡(P)-dim⁡coker(P)
= dim⁡ker⁡(P)-dim⁡ker⁡(P*).

Note: this is well defined as ker⁡(P) and ker⁡(P*) are finite-dimensional vector spaces, for P Fredholm.

  • •

    index(P*)=-index(P).

  • •

    index(P+K)=index(P) for any compact operator K.

  • •

    If P1:ℋ1→ℋ2 and P2:ℋ2→ℋ3 are Fredholm operators, then index(P2⁢P1)=index(P1)+index(P2).

  • •

    If t→Pt, t∈[0,1] is a norm continuous path of Fredholm operators, then index(Pt)=index(P0).

Fredholm operators of the form 𝑖𝑛𝑣𝑒𝑟𝑡𝑖𝑏𝑙𝑒+𝑐𝑜𝑚𝑝𝑎𝑐𝑡 have index zero.
